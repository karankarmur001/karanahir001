web

The Web is the common name for the World Wide Web, a subset of the Internet consisting of the pages that can be accessed by a Web browser.

www

all the public websites or pages that users can access on their local computers and other devices through the internet

browser

A browser is a software program used to locate and display information on the Internet or an intranet

addressbar

the familiar text field at the top of a web browser's graphical user interface (GUI) that displays the name or the URL (uniform resource locator) of the current web page.

titlebar

a graphical user interface (GUI) component of a software application or Web page

url/uri

URI is an acronym for a Uniform Resource Identifier. URL is an acronym for a Uniform Resource Locator.

http/https

Hypertext Transfer Protocol Secure (https) is a combination of the Hypertext Transfer Protocol (HTTP) with the Secure Socket Layer (SSL)/Transport Layer Security (TLS) protocol.

extention

an act or instance of extending, lengthening, stretching out, or enlarging the scope of something.

editor

a person who edits especially as an occupation.

network

a group of two or more devices or nodes that can communicate.

tcp/ip

Transmission Control Protocol/Internet Protocol

hosting

a service through which storage and computing resources are providing to an individual or organization for the accommodation and maintenance of one or more websites and related services.

domain

a particular field of thought, activity, or interest, especially one over which someone has control, influence, or rights.

server

a computer program or device that provides a service to another computer program and its user, also known as the client.

client

a person or company that receives a service from them in return for payment.
